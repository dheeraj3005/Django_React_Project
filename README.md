DJANGO-React Project
Outcome : To implement the CRUD operations using REST API using Django Framework(Python).

Backend :MacOS
Step1: Create Virtual Environment directory(To reduce conflcits) : python3 -m venv v
Step2: Create a activate Virtual Environment : source "path"/env/activate
Step3: Install all the dependencies from the requirements.txt using python : pip install -r requirements.txt
Step4: Using all the startup code to install django : python manage.py runserver (Follow this for startup of Django : https://docs.djangoproject.com/en/5.1/intro/tutorial01/)

Frontend:

Framework - React + vite
Create - npm create vite@latest my-react-app --template
startup - npm run dev
